Welcome to the ESDS Apps Server!

This server contains a number of applications built collaboratively on top of Dancecloud for use by the Edinburgh Swing Dance Society.

You can find the source code for this server here. If you're interested in collaborating, please contact info@esds.org.uk.

There are three pieces of content available:

  • Our membership cards management app.
  • Our Dancecloud POS permissions app.
  • Our rapid card scanner app.
